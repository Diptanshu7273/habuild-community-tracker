# WhatsApp Community Tracker Dashboard

An automation dashboard to monitor WhatsApp community member limits and prevent invite link overflow.

## Features
- Tracks all WhatsApp communities in real-time
- Displays member count of each group
- Warning alert when groups approach 1600+ members
- Automatic notification on WhatsApp and dashboard
- Shows Safe / Warning / Danger zones
- Search and filter communities
- CSV export support

## Tech Stack
- Node.js
- Express.js
- WhatsApp Cloud API
- HTML, CSS, JavaScript
- Real-time dashboard UI

## Purpose
WhatsApp communities have a limited member capacity.  
This tool automatically monitors community size and alerts admins to change the invite link before the group becomes full.
